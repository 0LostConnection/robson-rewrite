# Robson Templase - Rewrite

Rewriting [Robson Bot](https://github.com/0LostConnection/Robson-Bot) using ES6.

## How to use:

1. **Install dependencies:**
    ```bash
    npm install
    ``````
2. **Configure environment variables:**
    ```bash
    BOT_TOKEN=
    CLIENT_ID=
    DEBUG_GUILD_ID=
    ```
3. **Start the bot:**
    ```bash
    node index.js
    ```
